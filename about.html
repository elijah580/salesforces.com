About this project

Agentforce landing page clone

This About page explains the homepage structure, design decisions, and content strategy for the static Salesforce-inspired landing page. It uses clean HTML and CSS, with an emphasis on visual hierarchy, meaningful sections, and reusable components.

  • Static HTML/CSS website built for marketing and product storytelling.
  • Simple responsive navigation and content sections that mirror the homepage.
  • Local and online image sources used for visual examples.

What the homepage shows

Navigation

The top navigation uses a fixed header, responsive links, and a mobile-friendly dropdown to support both desktop and smaller screens.

Hero section

A clear hero section introduces the platform, highlights the value proposition, and offers two direct actions: start for free or watch a demo.

Feature highlights

Cards and content areas explain the platform’s AI capabilities, collaboration benefits, and customer success themes.

Industry use cases

Industry tiles show how the product would apply across finance, retail, healthcare, construction, education, and professional services.

Design and structure

Static layout

The page is fully static and depends only on HTML markup and CSS styling. Font Awesome provides iconography, while images are loaded locally or from trusted online sources.

Responsive behavior

Responsive CSS rules hide non-critical elements on narrow screens and ensure the header remains usable on mobile devices.

Visual hierarchy

Bold headings, consistent color usage, and whitespace create a strong information flow across the homepage sections.

Asset sources

Assets include local files from the `img/` folder and a curated online image sourced from Unsplash for professional visual context.
